# FitHub

## Description

Provide a short description explaining the what, why, and how of your project. Use the following questions as a guide:

The motivation for this app is for all users to find a fitness program and exercise that they can use, whether they are at home or at the gym to meet their fitness goals. Visualization helps demonstrates how to do the exercise correctly with written details for user to read and understand the exercises. This app solves the problem of figuring out what exercise to do for certain body parts when user is about to start the workout.
